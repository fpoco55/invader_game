# インベーダーゲーム

## 概要
シンプルなスペースインベーダー風ゲームです。プレイヤーは矢印キーで移動し、敵を避けながら生存時間を延ばします。

## システム要件
- Windows OS
- 1GB以上の空きディスク容量

## インストール方法
1. GitHubのリリースページにアクセス：https://github.com/fpoco55/invader_game/releases
2. `v1.0`リリースのページを開く
3. `invader_game_v1.0.exe`をダウンロード
4. ダウンロードした.exeファイルをダブルクリックして実行
5. ゲームを開始

## ゲームプレイ
- 左矢印キー: プレイヤーを左に移動
- 右矢印キー: プレイヤーを右に移動
- ゲームウィンドウを閉じるか、[Ctrl+C]を押してゲームを終了
